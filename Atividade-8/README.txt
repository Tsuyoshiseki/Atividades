# Progressão Aritmética (PA) - Java

Este programa em Java calcula a soma dos termos de uma **Progressão Aritmética (PA)**. A PA é uma sequência de números em que a diferença entre os termos consecutivos é constante.

---

## Funcionalidades

- Calcula a soma dos termos de uma PA a partir do primeiro valor, do segundo valor, e do montante fornecidos pelo usuário.
- Exibe o valor da soma dos termos no console.

---

## Fórmulas Utilizadas

A fórmula geral da Progressão Aritmética (PA) é:

- O termo geral: 

an = a1 + (n-1) * r

Onde:
- `a1`: o primeiro termo
- `r`: razão da PA (a diferença entre dois termos consecutivos)
- `n`: número do termo desejado.

- A soma dos `n` primeiros termos da PA é dada por:

Sn = (a1 + an) * n / 2
