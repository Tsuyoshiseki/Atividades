# Atividades de Programação

Este repositório contém uma coleção de atividades práticas que desenvolvi ao longo dos meus estudos. São exercícios variados que envolvem **lógica de programação**, **algoritmos**, **resolução de problemas** e desafios **sem uma linguagem específica padrão** — ou até mesmo com várias misturadas.

---

## Objetivo

Organizar e compartilhar resoluções de exercícios feitos durante:

- Cursos online e presenciais  
- Estudos na faculdade  
- Prática pessoal e testes de lógica  
- Desafios aleatórios de raciocínio
